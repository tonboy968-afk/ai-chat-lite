# 🚀 LM Studio Chat Lite

> 一个轻量级、开箱即用的本地AI聊天界面，让LM Studio的使用变得简单优雅。

## 🎯 解决的问题

LM Studio 作为本地大语言模型运行工具，虽然功能强大，但原生界面存在以下痛点：

- ❌ **界面复杂** - 配置参数多，新手难以上手
- ❌ **缺乏角色管理** - 无法快速切换不同场景的AI助手
- ❌ **功能分散** - 系统提示词、模型配置等设置不够直观
- ❌ **部署困难** - 需要额外配置才能在局域网内多设备访问

**LM Studio Chat Lite** 完美解决了这些问题！

## ✨ 产品特色

### 🎭 智能角色系统
- **7个预设角色** - 翻译助手、代码助手、创意写作等，一键切换
- **自定义角色** - 创建专属AI助手，保存系统提示词
- **角色持久化** - 自定义角色自动保存到浏览器，下次打开依然可用

### 📱 极简体验
- **单文件部署** - 一个HTML文件，双击即用，无需安装
- **响应式设计** - 完美适配手机、平板、电脑
- **流式响应** - 实时显示AI生成内容，打字机效果

### 🔧 灵活配置
- **多模型支持** - 兼容所有LM Studio加载的模型
- **局域网共享** - 一次配置，所有设备都能访问
- **本地存储** - 对话记录和设置保存在浏览器，隐私安全

### 🌐 开源免费
- **完全开源** - MIT协议，可自由使用和修改
- **零依赖** - 纯HTML/CSS/JavaScript，无需Node.js或服务器
- **跨平台** - Windows/Mac/Linux/手机都能用

## 🚀 快速开始

### 方法一：直接使用（推荐）
1. 下载 `index.html` 文件
2. 双击打开，即可在浏览器中使用
3. 在设置中配置你的LM Studio API地址

### 方法二：局域网共享
1. 将HTML文件放在任意文件夹
2. 使用Python启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 其他设备访问：`http://你的IP:8000/index.html`

## ⚙️ 配置说明

### LM Studio 设置
- **API地址**: LM Studio的API端点（默认：`http://localhost:1234`）
- **模型名称**: 留空使用默认模型

### 角色配置
- **系统提示词**: 定义AI的角色和行为
- **预设角色**: 7个常用角色，开箱即用
- **自定义角色**: 无限创建，保存到浏览器
